# SVG Logo Maker [![License: The MIT License](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
  ## Description
  Creates a 300 x 200 logo in .svg format based on a color, text, and a shape.
